# FinNudge AI 

FinNudge AI is an AI-powered personal finance management application that helps users track income, expenses, savings goals, budgets, and financial insights through a clean and intuitive interface.

## Features

-  JWT Authentication
-  Income & Additional Income Management
-  Fixed & Variable Expense Tracking
-  Savings Goals
-  Financial Analytics
-  AI Financial Nudges
-  Dashboard with Insights
-  Money Streak System
-  Supabase PostgreSQL Database

## Tech Stack

### Frontend
- Flutter

### Backend
- FastAPI
- SQLAlchemy

### Database
- PostgreSQL (Supabase)

## Installation

### Clone the repository

```bash
git clone https://github.com/sohamjambhale731-design/FinNudgeAI.git
```

### Backend

```bash
cd backend
python -m venv .venv
```

Activate virtual environment

Windows

```bash
.venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file inside the backend folder.

Run the backend

```bash
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
flutter pub get
flutter run
```
